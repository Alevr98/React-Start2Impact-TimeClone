# Start2Impact Times
Start2Impact Times è un clone dell'homepage del New York Times.
![Logo Start2Impact Times logo](https://start2impact-clone.netlify.app/static/media/logo.c39f9ea4559af2e48875.png "Logo Start2Impact Times")
Consente di visualizzare le ultime notizie disponibili e, con la funzione di ricerca è ora possibile trovare gli articoli più interessanti e i trend del momento.
